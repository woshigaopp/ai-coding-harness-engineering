# Tools

This directory is reserved for validators and workflow automation.

The methodology expects tools that can eventually enforce:

- artifact completeness;
- cross-reference integrity;
- module contract graph consistency;
- Atomic Issue self-containment;
- task DAG validity;
- pre-execution admission;
- task diff allowlists;
- task-local semantic review;
- backflow invalidation.

The current repository publishes the methodology and templates first. Tooling
can be added incrementally.
